Допущения при решении задачи по "трясучке".

1 В примере входных данных первой строкой идет N - число кадров, 
далее в следующей строке N-1 чисел - расстояний между прямоугольниками в соседних кадрах.
При этом в примере N=14, а после идут 14 чисел.
Считаем, что N - это уже число кадров минус 1.

2 В качестве выходных данных надо определить минимальное число подрядидущих кадров K, 
для которого сумма расстояний превышает 1000. И приведено число 3.
То есть три подряд идущих значения входной последовательности должны дать сумму, 
превышающую 1000. 
Но во входной последовательности максимальное значение суммы 3-х подряд идущих значений = 1000.
Таким образом, для того, чтобы получить искомое 3, ищем 
минимальное число подрядидущих кадров K, для которого сумма расстояний РАВНА или превышает 1000.
